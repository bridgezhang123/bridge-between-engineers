## 2. 标准引用

- GB/T 8013.1-2018 铝及铝合金阳极氧化膜与有机聚合物膜 第1部分：阳极氧化膜
    - 规定了铝及铝合金阳极氧化膜的术语定义、分类、性能要求、试验方法及检验规则。
    - 按应用场景将阳极氧化膜分为**装饰性**和**防护性**两大类。并定义了详细的性能指标体系，如膜厚、封孔质量、耐磨性、耐蚀性等。
    - 明确不适用于铬酸溶液中阳极氧化生成的膜等场景。
- GB/T 19822-2024 铝及铝合金硬质阳极氧化膜规范
    - 通过特殊的阳极氧化工艺（通常是低温、高电压、高电流密度）在铝及铝合金表面生成的一种厚度厚、硬度高、耐磨性好的氧化膜，主要服务于工程和功能性应用场景。
    - 该标准对供需双方的信息交互（需方应向供方提供的信息，见标准附录A）提出了明确要求，强调了根据应用需求定制工艺方案的重要性。
    - 鉴于其**工艺窗口**（温度、电压）比普通阳极氧化更窄，对合金种类和加工质量的要求也更高。
- MIL-PRF-8625F(2) NOT 1 Anodic Coatings for Aluminum and Aluminum Alloy
    - 美国军用标准MIL-A-8625及其后续修订版MIL-PRF-8625。区分普通硫酸阳极氧化（Type II）与硬质阳极氧化（Type III）比较通用的命名体系。其[官网检索入口](https://quicksearch.dla.mil)可检索到并提供下载。

## 3. 实操与模板

### 3.1 背景介绍

- 阳极氧化本质上是一种电化学过程，将铝合金制品作为阳极置于特定电解液中，通过电解，在表面原位生成一层致密的氧化铝（Al₂O₃）膜。
- 结构与外观：氧化膜具有蜂窝状的多孔结构，使其能强力吸附染料，实现丰富色彩；而未经染色的本色氧化膜则能保留金属质感。
- 从第二节的标准可知，阳极氧化有其成熟、完善的标准体系：
    - 核心工艺参数：氧化温度、氧化电压、氧化电流等。
    - 评价维度：膜厚、封孔质量、耐磨性、耐蚀性等。

### 3.2 问题复盘

- 现象：6061-T6 黑色阳极氧化壳体浸泡海水约21天后出现虹彩状油斑。
- 位置：主要集中在壳体中部及端盖边缘。
- 可逆性：湿布擦拭后暂时消失，干燥后重新显现。
- 技术判断：最可能原因是封孔不良、封孔前清洗不彻底或染色剂残留。
	- 微孔中残留封孔液、染色液、杂质或金属离子；
	- 在海水电解质作用下，残留物逐步迁移到表面，形成虹彩油斑。
	- 说明：单次“同工艺重做”通常难以根治，必须改进工艺参数和清洗/封孔流程。

- **Edgetech 3400 参考**
	- 美国 edgetech 公司的 `3400 Sub-Bottom Profiler Electronics Bottle and End Cap` 标明为铝合金6061-T6 hardcoat anodized(手册PDF第81页，Figure 2-29: 3400 Sub-Bottom Profiler Electronics Bottle ICD)，见公开资料 [`EdgeTech 3400` 官网手册](https://www.edgetech.com/wp-content/uploads/2023/04/0021270_Rev_F.pdf)，存档于[互联网档案馆](https://web.archive.org/web/20250318115004/http://www.edgetech.com/wp-content/uploads/2023/04/0021270_Rev_F.pdf)。
	- 该案例说明：海洋电子舱体等海水浸泡部件，行业首选硬质阳极氧化（hardcoat anodize）.

### 3.3 解决措施

网上能检索到如下大致信息：

| 对比维度 | 普通阳极氧化 (Type II) | 硬质阳极氧化 (Type III / Hardcoat) |
|---|---|---|
| 工艺温度 | 室温或略低（约20–25°C） | 低温（约0°C） |
| 膜厚（典型） | 5–25 µm | 25–100+ µm |
| 硬度 | 中等 | 极高（耐磨） |
| 孔隙率/致密性 | 较高（多孔） | 较低（致密） |
| 耐盐雾/耐蚀性能 | ~96–120 小时（典型） | ~500–1000+ 小时（典型） |
| 主要用途 | 装饰、轻度防护 | 严酷环境、耐磨与海水应用 |

- **黑色普通阳极氧化**：工艺：硫酸阳极化 + 依赖染色成黑 + 封孔；风险：封孔不良或前处理清洗不彻底时，残留封孔剂、染色剂、杂质会在海水中缓慢渗出，出现虹彩油斑。
- **硬质阳极氧化（hardcoat anodize）**：虽仍需封孔，但整体膜层致密性更高，对微孔残留物敏感度更低，海水环境下稳定性更好。

## 4. 其余要点

- 尽管有详尽的标准，但在和加工商/氧化厂沟通时，尤其是小的氧化厂，他们可能不会严格按照标准执行阳极氧化；真正专业的氧化厂对订单的量、总金额一般会有门槛。
- 防护级或是硬质阳极氧化，对于核心工艺参数，氧化厂可能并不一定会提供；此时，对评价维度：膜厚、封孔质量、耐磨性、耐蚀性等进行要求是更合理的选项。
- 在批量氧化前，申请样件或是打样进行测试可能是必要的流程。
