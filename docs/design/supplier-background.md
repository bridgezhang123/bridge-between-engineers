# 总览

- 考虑时间成本，寻找可能的优质供应商、产品目录。

## 综合类供应商

- [misumi](https://www.misumi.com.cn/)、[怡合达](https://www.yiheda.com/)，提供标准件、自动化模组、非标准件加工等产品/服务；分别为日资全球化领先企业与国内上市对标企业，标准化程度很高。

## 以国标O形圈举例

- misumi、怡合达均有符合国标(GB/T 3452.3-2005)的产品；淘宝有的店铺有销售O形圈规格(236*3.55)为非国标内径系列，询问店家竟然回复他们符合国标，其页面内容专业性存疑，不禁让人产生较深的困惑。
- 同规格下，misumi、怡合达的报价是上述淘宝店铺商品的10倍甚至更高。
- 显然，从长期而言，选择符合国标的产品是对产品质量、长期视角下的经济性的必然选择。
