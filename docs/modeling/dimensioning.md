# 基本标注

> 标注的核心不是把尺寸写上去，而是让加工、检验与装配都能据此稳定执行。

## 1. 范围与目标

本文主要讨论：

- 基本尺寸标注的原则
- 公差与配合的基本表达
- 如何减少重复、遗漏和歧义

## 3. 实操与模板

### 3.1 基本要求

- 优先标注功能尺寸
- 尺寸链尽量清晰，避免不必要的重复闭环
- 尺寸应服务于加工与检验，而不只是几何表达

### 3.2 螺纹孔与孔标注

对螺纹孔、均布孔和贯穿孔，应尽量采用软件中稳定、可复用的标注方式，减少手工录入错误。

### 3.3 公差配合

- 孔轴配合应明确采用何种配合代号
- 重要配合不应只写名义尺寸
- 对未注公差与关键尺寸公差，应有清楚边界

### 3.4 可读性

- 尺寸线、引出线和注释不应相互干扰
- 均布孔、螺纹孔和贯穿特征宜采用稳定写法
- 必要时用局部视图、剖视图和详图辅助表达

## 4. 其余要点

### 4.1 自审核清单

- 是否符合三视图、相视图的视图布局要求。
- 辅助标记是否完整(中心线、圆心标记、剖面线等)。
- 对于通孔，若剖面能体现通孔特征最好，否则宜加注“贯穿”二字。
- 是否符合可读性(尺寸线、引出线和注释不应相互干扰等)的要求。
- 是否遵守了零件的尺寸链的基本要求进行标注。
- 

### 4.2 检验视角

尺寸不是只给设计人员看的，也应站在加工和检验角度复核一遍：这条尺寸能否测、是否容易误解、是否真的重要。

## 5. 边界与风险

- 过度依赖自动标注，容易出现信息堆叠
- 公差标注若脱离实际加工能力，容易形成纸面正确、制造困难
- 仅靠未注公差覆盖所有情况，通常不够稳妥

## 6. 小结

标注工作的重点，不在于把每个尺寸都写出来，而在于让真正重要的尺寸表达清楚、可制造、可检验。图纸越清楚，后续沟通成本越低。
