# Drawing Template / 图纸模板

> 图纸模板不是装饰层，它决定了标题栏、字体、未注公差栏和页幅策略是否能长期统一。

## 1. Scope and Goal / 范围与目标

本文主要讨论图纸模板应当固定哪些基础信息，以及个人或尚未形成成熟体系的小团队，如何先搭出一套可长期复用的模板基础。

重点不在于一次性做得很复杂，而在于先把标题栏、页幅、字体、未注公差与基本识别信息稳定下来，减少后续每次出图时的重复设置。

## 3. Practice and Templates / 实操与模板

### 3.1 Fixed Elements / 建议固定的内容

一套稳定的图纸模板，通常建议优先固定以下内容：

- 标题栏基础字段
- 常用页幅
- 字体与字号
- 未注公差说明
- Logo、名称与网址等识别信息

这些内容一旦固定，后续不同图纸之间的阅读体验会更一致，也更方便后续统一维护。

### 3.2 Template Strategy / 模板策略

对于个人或小团队，建议先从最常用的模板开始，而不是一开始维护很多套版本。通常可先固定：

- `A4` 竖版模板
- `A3` 横版模板

先把高频场景跑顺，再考虑是否扩展到其他页幅或特殊模板，会更现实。

### 3.3 What the Template Should Carry / 模板应承载什么

模板负责统一基础表达，但不替代具体设计判断。比如：

- 标题栏负责承载项目、图号、版本等基础信息
- 未注公差栏负责承载默认规则
- Logo、名称和网址可作为来源识别与适度引流入口

如果模板承担了过多本该由设计者逐项确认的内容，后续反而容易出错。

## 4. Other Key Points / 其余要点

### 4.1 Title Block and Tolerance Area / 标题栏与未注公差区

图纸模板里最值得认真设计的，不只是外观，而是标题栏和未注公差区的表达顺序。它们决定了读图者能否快速找到关键信息。

如果后续补图，我建议您在这里插入一张模板截图，重点标出：

- 标题栏主要字段
- 未注公差位置
- Logo、名称与网址的放置区域

### 4.2 Branding Should Be Moderate / 识别信息应适度

模板中加入 Logo、名称和网址是可以的，但应保持克制。它们应当起到识别与来源说明作用，而不应盖过图纸主体信息。

## 5. Boundaries and Risks / 边界与风险

- 模板字段过多，会增加维护成本
- 未注公差栏不应成为省略关键尺寸要求的理由
- Logo、名称或网址过于抢眼，会干扰读图
- 模板统一并不等于所有图纸都能不加判断地直接套用

## 6. Summary / 小结

图纸模板的价值，不在于“做得多漂亮”，而在于它是否让标题栏、页幅、字体和未注公差这些基础表达长期稳定下来。

对个人或小团队而言，先把少量高频模板做稳，比一次性做出很多版本更有实际意义。
