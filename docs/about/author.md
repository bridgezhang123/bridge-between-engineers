# About Author / 关于作者

## Brief Profile / 简要说明

张桥 / Bridge Zhang，机械工程师，毕业于杭州电子科技大学机械工程学院。

这个页面并不想写成一份强调展示感的个人简介，更接近一份简要的作者说明。相较于“说了多少”，我更在意是否把问题想清楚、把内容写准确，也尽量把经验整理成便于复盘与复用的形式。

## Current Focus / 当前关注

目前，我主要围绕机械设计、建模出图、制造、测量与检验等方向持续学习，并把其中较有整理价值的内容逐步写成文档。

这些内容一方面服务于我自己的回看与反思，另一方面，如果也能对同行或后来者提供一点参考，我会觉得这项整理工作是值得长期做下去的。
